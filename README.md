# Bank-Customer-Churn-Predictor

A Full-Stack Data Science Solution for Proactive Retention

## 📌 Project Overview
This project is an end-to-end Machine Learning application designed to help bank relationship managers identify "at-risk" customers. Unlike standard churn models, this tool integrates Explainable AI (SHAP) and Financial ROI Logic to turn raw predictions into actionable business strategies.

## 🛠️ The Technical Stack
Language: Python 3.11+

Machine Learning: Scikit-Learn (Random Forest Classifier)

Explainability: SHAP (Shapley Additive Explanations)

Web Framework: Streamlit

Deployment: GitHub Desktop & Streamlit Cloud

## 🚀 Key Features
Real-time Prediction: Instant churn probability based on customer demographics.

Explainable AI: A visual breakdown of why a customer is likely to leave (Age, Balance, etc.).

Strategic ROI Calculator: Estimates the cost of intervention vs. the potential profit saved.

Mobile Responsive: Optimized for use on smartphones during offline consultations.

## 📊 How to Run Locally
Clone the repo: git clone https://github.com/AbhinandanThatikonda/Bank-Customer-Churn-Predictor.git

Install dependencies: pip install -r requirements.txt

Run the app: streamlit run app.py
